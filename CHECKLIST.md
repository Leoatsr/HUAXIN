## ⚠️ 失败项的紧急修复

### 白屏
→ F12 Console 看报错。多半是 `build/` 目录没 npm install 就推了。

### Manifest 报错
→ 检查 `public/manifest.webmanifest` 是否上传，打开 `https://你的域名.com/manifest.webmanifest` 应返回 JSON。

### SW 没注册
→ SW **只在 production 生效**（不在 dev）。确保你推的是 `npm run build` 产物，不是 `npm run dev` 的热更新。

### og-image 打不开
→ 检查 `public/og-image.png` 是否在仓库里（34 KB 左右）。

### 音频不响
→ 浏览器**政策**：用户没交互前不能播音频。先点一下地图或任何按钮，再点音乐按钮。
