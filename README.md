# 花信风 🌸

> 跟着天地节律，追一场中国色

一幅活着的中国物候地图。基于真实积温模型和气象数据，动态追踪全国各地花卉、树木的观赏状态。

## ✨ 特性

- **真实中国地图** — D3.js + 阿里云 DataV GeoJSON 渲染完整中国省级行政区划
- **真实积温模型** — 接入 Open-Meteo 气象API，计算各地 ≥5°C 有效积温，动态判定物候状态
- **四季花事** — 覆盖春樱、夏荷、秋枫、冬梅等 30+ 个赏花/赏叶胜地
- **花卉专属图标** — 每种花草树木都有手绘 SVG 矢量图标（樱花、牡丹、杜鹃、荷花、银杏、红枫...）
- **卷轴式交互** — 开场拖动卷轴展卷入画，应季诗词作为引导
- **古韵背景音** — Tone.js 程序化生成宫商角徵羽五声音阶旋律
- **多维度操控** — 鼠标滚轮/触摸板双指缩放、方向键/拖动平移、区域快速导航

## 🛠 技术栈

- **React 18** — UI 框架
- **D3.js** — 地理投影与地图渲染
- **Tone.js** — Web Audio 音乐引擎
- **Vite** — 构建工具
- **Open-Meteo API** — 气象数据源
- **DataV GeoJSON** — 中国地图数据源

## 📖 物候模型

采用 **Spring Warming 模型**（春季积温模型）：

- 自每年 1月1日 起，累计日均气温 ≥5°C 的有效积温
- 当累计值达到该物种历史开花所需阈值时，判定为盛花期
- 阈值因物种和海拔而异（如梅花 200°C·d，牡丹 320°C·d）
